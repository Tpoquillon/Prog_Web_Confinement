Bienvenue sur StageBook !

Qu'est-ce que ce site ?

StageBook est un site web répertoriant les stages effectués par les élèves de Biosciences. Il est né d'une volonté de créer une base de données ressource facile d'accès pour les élèves lors de leur propre recherche.

Architecture générale de la page

Le site s'articule en 5 pages. Vous vous trouvez actuellement sur l'index, c'est-à-dire la page d'accueil. D'ici sont accessibles les pages "A Propos", "Stages", et "Recherche Avancée".
